# Github Repo Search
DEMO: [http://purepennons.github.io/github-repo-search](http://purepennons.github.io/github-repo-search)

## Development
### Application
```shell
$ yarn && yarn start
```
Open [http://localhost:3000/](http://localhost:3000/).

### Component Preview
```shell
$ yarn styleguide
```
Open [http://localhost:6060/](http://localhost:6060/).

## Tech Stack
* react
  - 16.3 context api
* apollo-link-rest
* styleguidist
